# FWDNXT SDK manual

## Hardware supported: 
This SDK supposes that you are working on a desktop computer with Micron FPGA boards on a PCI backplane (AC-510 and EX-750 for example).

## Operating systems supported:
* Ubuntu 16.04
* CentOS 7
